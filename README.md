# Library Management System

##  Overview
This is a simple and fast Library Management System built using Python.  
It allows users to add books, search books, issue and return books.  
The project demonstrates basic file-handling, data structures, and modular programming.

##  Features
- Add new books  
- View all books  
- Search for a book  
- Issue a book  
- Return a book  
- Simple menu-driven interface  

## Technologies / Tools Used
- Python  
- GitHub  
- VS Code / IDLE (optional)

##  How to Run the Project
1. Install Python 3  
2. Download or clone the repository
